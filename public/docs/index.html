Introduction

This is Laravel project which has used following features like guards,spatie laravel-permission, yajra-datatable,laravel-passport for api

Base URL: http://127.0.0.1:8000/

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

To authenticate requests, include an Authorization header with the value "Bearer your-token".

All authenticated endpoints are marked with a requires authentication badge in the documentation below.

You can retrieve your token by login Generate API token.

Authentication

Log out the authenticated user.

requires authentication

Example request:
curl --request POST \
    "http://127.0.0.1:8000/api/logout" \
    --header "Authorization: Bearer abEc1Dkh368aVe64dfgvP5Z" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://127.0.0.1:8000/api/logout"
);

const headers = {
    "Authorization": "Bearer abEc1Dkh368aVe64dfgvP5Z",
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "POST",
    headers,
}).then(response => response.json());

Example response (200):


{
    "success": true,
    "message": "Logged out successfully.",

}

Base64 to Image

Save an image from base64 encoded string.

requires authentication

Example request:

    "http://127.0.0.1:8000/api/image" \
    --header "Authorization: Bearer 8bDZ16da6EVvachkf345gPe" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"image\": \"quibusdam\"
}"

const url = new URL(
    "http://127.0.0.1:8000/api/image"
);

const headers = {
    "Authorization": "Bearer 8bDZ16da6EVvachkf345gPe",
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "image": "quibusdam"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());
