Introduction

This is Laravel project which has used following features like guards,spatie laravel-permission, yajra-datatable,laravel-passport for api

Base URL: http://127.0.0.1:8000/

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

To authenticate requests, include an Authorization header with the value "Bearer your-token".

All authenticated endpoints are marked with a requires authentication badge in the documentation below.

You can retrieve your token by login Generate API token.

Authentication

Log out the authenticated user.

requires authentication

Example request:
curl --request POST \
    "http://127.0.0.1:8000/api/logout" \
    --header "Authorization: Bearer 1P8Zc5ehEf6gd6baD3kaV4v" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://127.0.0.1:8000/api/logout"
);

const headers = {
    "Authorization": "Bearer 1P8Zc5ehEf6gd6baD3kaV4v",
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "POST",

}).then(response => response.json());

    --header "Accept: application/json" \

    \"image\": \"dolore\"
}"

    "http://127.0.0.1:8000/api/image"
);

};
